# Day 5: Final DSA Practice

**Time:** ~60 minutes

**Practice file:** Create `day5_practice.py`

For every challenge:

1. State input and output.
2. Give a simple approach.
3. Write and test the solution.
4. State time and space complexity.

---

## Challenge 1: Two Sum

Solve it first with nested loops, then with a dictionary.

## Challenge 2: Valid Parentheses

Use a stack.

## Challenge 3: Binary Tree Level Order

Use BFS and a queue.

## Challenge 4: Number of Islands

Use DFS or BFS and mark visited land.

## Challenge 5: Kth Largest

Use a heap of size `k`.

## Challenge 6: Maximum Window Sum

Use a fixed-size sliding window.

---

## Final Checklist

You are ready for interview-pattern practice when you can:

- Compare brute force with an optimized solution
- Choose lists, dictionaries, sets, stacks, queues, trees, or heaps
- Write binary search
- Traverse a tree with DFS and BFS
- Recognize two pointers and sliding windows
- State basic time and space complexity
- Test empty, small, normal, and tricky inputs

Do not worry about solving every problem quickly. Clear thinking comes first;
speed arrives through repetition.
