# Day 5: Week 7 Practice — Tests and Files

**Time:** ~50 minutes

**Practice file:** Create `day5_practice.py`

---

## Challenge 1: Testable Grade Helper

Write:

```python
def letter_grade(score):
    ...
```

Return:

- `A` for 90+
- `B` for 80–89
- `C` for 70–79
- `D` for 60–69
- `F` below 60

Write pytest tests.

## Challenge 2: Todo File

Create a small todo program:

1. Add task
2. List tasks
3. Quit

Save tasks in `todo.txt`.

## Challenge 3: CSV Analyzer

Given `scores.csv`, print:

- number of students
- highest score
- average score

## Challenge 4: Test Your CSV Logic

Move average-score logic into a function and test it with `assert` or pytest.

## Graduation Check

You should be able to explain:

- why functions are easier to test than print-only scripts
- why `with open(...)` is preferred
- when to use text files versus CSV
