# Day 5: Week 8 Practice — Regex and OOP

**Time:** ~55 minutes

**Practice file:** Create `day5_practice.py`

---

## Challenge 1: Email Validator

Write `is_valid_email(email)` using regex.

Accept simple emails like:

```text
sagar@example.com
name123@test.org
```

## Challenge 2: Phone Number Extractor

Extract all phone numbers shaped like:

```text
123-456-7890
```

## Challenge 3: Contact Class

Create a `Contact` class with:

- name
- email
- phone
- `__str__`

Validate email with regex.

## Challenge 4: Contact Book

Store multiple contacts in a list.

Add functions to:

- add contact
- search by name
- list all contacts

## Graduation Check

You should be able to explain:

- why regex is useful for validation and extraction
- what `self` means
- why `__init__` exists
- when a class is worth creating
