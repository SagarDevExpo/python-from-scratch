# Python Intermediate — Weeks 6–9

This section fills the main CS50P-style topics that come after Python basics
and before deeper DSA.

Everything is self-contained. You do not need to jump to other folders.

## How to Use

For every day:

1. Read the `.md` lesson.
2. Create the requested `dayX_practice.py` yourself.
3. Type examples from memory when possible.
4. Run your code often.
5. Finish the exercises before moving on.
