## Rules
1. **Type every line yourself** — don't copy/paste
2. **Run the code after every section** — see what happens
3. **Do the exercises** — reading alone doesn't work
4. **If stuck for 10 min, move on** — come back tomorrow
5. **Practice days are NOT optional** — that's where learning happens

## How to Run
```bash
cd python-from-scratch/week1
python3 day1_print_variables.py
```
